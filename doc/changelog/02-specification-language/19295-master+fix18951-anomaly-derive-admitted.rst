- **Changed:**
  The syntax :n:`Derive x SuchThat type As name` is discouraged and replaced by
  :n:`Derive x in type as name`
  (`#19295 <https://github.com/coq/coq/pull/19295>`_,
  by Hugo Herbelin).

- **Added:**
  :cmd:`Derive` now supports the more general syntax
  :n:`Derive open_binders in type as name`
  (`#19295 <https://github.com/coq/coq/pull/19295>`_,
  by Hugo Herbelin).
